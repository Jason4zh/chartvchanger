# Phigros工具

一个基于纯前端实现的Phigros谱面修改工具，包含**按键替换器**和**流速修改器**两个核心功能，所有操作均在浏览器本地完成，不会将文件上传至服务器，保障数据安全。

## 功能说明

### 1. 按键替换器 (converter.html)
将Phigros谱面（ZIP格式）中的按键类型批量转换，支持以下类型互转：
- Tap（点击）
- Drag（拖动）
- Hold（长按）
- Flick（滑动）

额外特性：
- 可选择是否包含Hold按键（转换非Hold类型时）
- 转换为Hold类型时，可自定义长按最小时长（毫秒）
- 实时显示加载的谱面文件名和按键总数
- 转换过程有详细的控制台日志（便于调试）

### 2. 流速修改器 (speed.html)
修改Phigros谱面的流速，支持自定义速度倍数：
- 自动识别谱面原始流速
- 实时预览流速效果（下落动画）
- 精确控制速度倍数（支持小数）
- 生成修改后的ZIP谱面文件

## 特点
- 🚀 纯前端实现：基于JSZip解析/生成ZIP文件，无需后端服务
- 📱 响应式设计：适配不同屏幕尺寸的设备
- 🔒 本地处理：所有文件操作均在浏览器内完成，不上传至任何服务器
- 🎨 简洁UI：操作流程清晰，交互友好

## 技术栈
- HTML/CSS（原生样式，无框架）
- JavaScript（原生ES6+）
- JSZip（用于ZIP文件的解析与生成）

## 使用方法

### 环境要求
- 现代浏览器（Chrome、Firefox、Edge、Safari等）
- 无需安装额外插件，直接打开HTML文件即可

### 通用操作
所有功能均需先上传Phigros谱面文件（ZIP格式，内含谱面JSON文件），支持：
- 点击上传区域选择文件
- 拖拽文件至上传区域

### 按键替换器使用步骤
1. 打开 `index.html`，点击「按键替换器」进入功能页面
2. 上传Phigros谱面ZIP文件
3. 选择目标按键类型（Tap/Drag/Hold/Flick）
4. 根据需要配置Hold相关参数（如最小时长）
5. 点击「下载转换后的ZIP文件」获取修改后的谱面

### 流速修改器使用步骤
1. 打开 `index.html`，点击「流速修改器」进入功能页面
2. 上传Phigros谱面ZIP文件
3. 在输入框中填写速度倍数（如`1.5`表示1.5倍速，`0.8`表示0.8倍速）
4. 实时预览区域会显示流速效果
5. 点击「下载修改后的ZIP文件」获取修改后的谱面

## 注意事项
1. 仅支持Phigros谱面的ZIP格式文件，且ZIP内必须包含JSON格式的谱面文件
2. 转换/修改后的文件命名规则：`原文件名_类型/倍数.zip`（如`test_tap.zip`、`test_1.50x.zip`）
3. 建议备份原始谱面文件，避免修改后无法恢复
4. 若谱面JSON文件结构非标准Phigros格式，可能导致解析/修改失败

## 运行方式
1. 克隆或下载本项目所有文件到本地
2. 直接用浏览器打开 `index.html` 文件
3. 根据需要选择对应的工具功能即可
